# Calendar WPA

## Prerequisites
- Node.js 8+
- TypeScript 2.7+
- [Nat 1.0+](git@github.com:naturali/nat.git)

## Installation
```bash
npm install
```
or 
```bash
yarn
```

## Develop
```bash
1. git clone git@github.com:naturali/nat.git && cd nat && npm link
2. cd [repository path]/desktop
3. npm run dev
4. npm run server
```
5. visit [http://localhost:3000/](http://localhost:3000/)

## Build
1. npm run build
